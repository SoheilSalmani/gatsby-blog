1. Check which SDKs and runtimes are currently installed.

   <Solution>

   ```shell
   dotnet --list-sdks
   dotnet --list-runtimes
   ```

   </Solution>

2. Create a console app using the `dotnet` CLI. Then, build and execute the app.

   <Solution>

   ```shell
   dotnet new console
   dotnet run
   ```

   </Solution>

3. Write the traditional "Hello World" app in C#. Then, transform it to a
   top-level program.

   <Solution>

   ```csharp nu
   using System;
   class Program
   {
       static void Main(string[] args)
       {
           Console.WriteLine("Hello World!");
       }
   }
   ```

   ```csharp nu
   using System;
   Console.WriteLine("Hello World!");
   ```

   </Solution>

4. Write the command to open the official documentation in a browser window for
   the `dotnet new` command.

   <Solution>

   ```shell
   dotnet help new
   ```

   </Solution>

5. Get help output at the command line..

   <Solution>

   ```shell
   dotnet new console -h
   ```

   </Solution>

6. In Visual Studio Code, what is the keyboard shortcut for _Go To Definition_.

   <Solution>
     It's <Kbd>F12</Kbd>.
   </Solution>
