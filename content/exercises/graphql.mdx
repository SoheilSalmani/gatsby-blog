1. Cr√©er un serveur GraphQL permettant d'afficher des informations personnelles
   (e-mail, URL de votre avatar, liste d'amis) vous concernant.

   <Solution>

   ```js
   const { ApolloServer, gql } = require("apollo-server")

   const typeDefs = gql`
     type User {
       email: String!
       avatar: String
       friends: [User]!
     }

     type Query {
       me: User!
     }
   `

   const resolvers = {
     Query: {
       me() {
         return {
           email: "soheil@salmani.me",
           avatar: "http://soheil.salmani.me/static/avatar.png",
           friends: [],
         }
       },
     },
   }

   const server = new ApolloServer({ typeDefs, resolvers })

   server.listen(4000).then(() => console.log("on port 4000"))
   ```

   </Solution>
