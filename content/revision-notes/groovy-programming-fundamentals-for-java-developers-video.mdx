## Getting Set Up

### Installing Groovy on a Mac or Linux System from the sdkman.io Website

Install SDKMAN!.

```shell
sdk ls groovy
sdk i groovy
sdk i groovy 4.0.0-alpha-3
sdk d groovy
groovy --version
```

```shell
sdk u 4.0.0-alpha-3
groovy --version
```

```shell
ls -l ~/.sdkman/candidates/groovy/
```

```shell
sdk ls
sdk help
```

### Creating and Debugging a Groovy Project in IDEA

```groovy nu fp=src/com/javaeasily/demos/HelloWorld.groovy
    }
}
```
